## A collection of my AutoHotKey projects.

- **chicky_search**: A custom popup search engine that takes in an array of options. Supports sets. Uses greedy fuzzy sort and bottom up merge sort for searching/sorting, so should actually be incredibly efficient on larger datasets. Saving a large dataset to a CSV/txt file then reading it into an array could be a great way to have large search sets.
- **file_helpers**: A collection of methods to create, delete, and modify text files to store arrays. Useful to maintain modifiable array values accross hotkeys (like if you have a hotkey that adds values to an array that you want to persist after exiting the hotkey). Supports arrays of both Strings and Objects.
