## A collection of my AutoHotKey projects.

- **search_engine (affectionately Chicky Search)**: A custom popup search engine that takes in an array of options. Supports sets. Uses greedy fuzzy sort and bottom up merge sort for searching/sorting, so should actually be incredibly efficient on larger datasets. Saving a large dataset to a CSV/txt file then reading it into an array could be a great way to have large search sets.
